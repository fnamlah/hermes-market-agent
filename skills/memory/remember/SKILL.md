# remember — Ingest into Cognee

Sends a text blob into Cognee's 3-store architecture (SQLite + LanceDB + Kuzu, embedded at `/data/cognee/`). Cognee runs LLM-based entity extraction + embeddings on every call — so use this **sparingly**, only on high-value outputs.

## When to use this skill

- **After daily-brief runs** — tag `daily-brief-YYYY-MM-DD`, ingest the brief text
- **After trader-reasoning on a high-priority ticker** — tag `trader-<TICKER>-YYYY-MM-DD`, ingest the JSON output
- **Manual** — when the user says "remember this: <fact>"

## When NOT to use this skill

- **Never on raw_posts** — would blow the budget
- **Never on every ticker_signals row** — same problem
- **Never on user greetings** or conversational noise

## Activation check

Run `python3 /app/shared/cognee_setup.py status` first. If it reports "not configured", tell the user:

> Memory (Cognee) is disabled. To enable, add `OPENAI_API_KEY` as a Railway service variable. Expected cost: $1-3/mo at our ingestion cadence.

Do not proceed until configured.

## How to invoke

```bash
cat /tmp/daily-brief.txt | python3 /app/shared/cognee_setup.py remember "daily-brief-2026-04-20"
```

Or via heredoc when the text is generated by another skill:

```bash
python3 /app/shared/cognee_setup.py remember "trader-NVDA-2026-04-20" <<'TEXT'
{full trader-reasoning JSON here}
TEXT
```

### Exit codes

- `0` — ingested successfully
- `1` — cognify() failed (network, LLM error) — transient, worth retrying once
- `2` — no stdin input
- `3` — OPENAI_API_KEY missing — report to user, do not retry
- `4` — cognee package missing (image build issue) — escalate

## Tag naming convention

- `daily-brief-<YYYY-MM-DD>` — one per trading day
- `trader-<TICKER>-<YYYY-MM-DD>` — one per high-priority trader-reasoning call
- `manual-<short-slug>` — user-initiated "remember this" facts

Using consistent tags lets the `recall` skill filter by dataset when appropriate.

## Cost guardrails

- Daily brief is ~500-800 words. cognify() on that = ~2K input tokens + ~1K output extraction = ~$0.002 with gpt-4o-mini.
- Trader-reasoning JSON is ~300 words. Similar cost.
- Expect $1-3/month at expected cadence. Monitor via OpenAI dashboard — not OpenRouter (Cognee uses OpenAI directly).

## Failure modes to handle

- `OPENAI_API_KEY` missing → exit 3 → tell user to set it
- Network error on cognify() → retry once after 5s; on second failure, log and skip
- Huge input (>10KB) → truncate to 10KB with a note (prevents runaway costs on accidental large inputs)

## What NOT to do

- Do not ingest the same tag+content twice — Cognee doesn't auto-dedup, so you'd double-count in vector similarity. If you need to update a fact, use a new tag with a version suffix.
- Do not invoke `cognify()` directly; always go through `cognee_setup.py remember` so the guard and path setup happen.
